# ⚔️ VS Arena — Weapon Battle Simulator

쇼츠에서 유행하는 물리 배틀 시뮬(무기 볼 배틀)을 브라우저에서 직접 돌려보는 웹 시뮬레이터.
매치업을 짜고 → 전투를 돌리고 → **링크 하나로 같은 경기를 공유**한다 (같은 시드 = 같은 전투).

## 실행 방법

정적 사이트라 서버 아무거나 붙이면 된다:

```bash
cd vs-arena
python -m http.server 8317
# 브라우저에서 http://localhost:8317 접속
```

배포는 GitHub Pages에 그대로 올리면 끝 (빌드 과정 없음).

## 기능

- **두 가지 모드**: 1v1 듀얼 + 배틀로얄(2~6명 난투, 마지막 1인 생존)
- 배틀로얄 처치 시 레벨업(성장): 킬하면 몸집·체력·데미지 상승 → 스노우볼 (With Healthbars류)
- 파이터 11종 자동 전투 — 타격할 때마다 무기/투사체가 자라는 밈 메커니즘
- 근접 8종: 곡괭이(3타 크리), 낚싯대(끌어당김), 검(콤보), 도끼(분노+가속), 해머(스턴+넉백저항), 트라이던트(흡혈), 드렁크가이(비틀거림+회피), 그래비티가이(중력장)
- 원거리 3종: 아처(리드샷 저격+카이팅), 닌자(수리검 3연발), 봄버(포물선 폭탄+광역폭발+자폭리스크)
- 회전 무기는 투사체를 쳐내는 방패 역할 (폭탄은 되받아침), 원거리는 근접거리에서 발사 불가
- WebAudio 합성 사운드 (무기별 타격음, 뮤트 토글) — 오디오 파일 없음
- GoatCounter 애널리틱스 (쿠키 없음, 프라이버시 친화)
- HP / 속도 / 시드 커스터마이즈
- 결정론적 시뮬레이션: 같은 시드는 항상 같은 경기 → 공유 링크로 리플레이 재현
- 공유 URL 예시: `/?a=hammer&b=trident&hp=100&spd=1&seed=7` (열면 자동 재생)
- 9:16 세로 아레나, 모바일 대응, 광고 슬롯 자리 예약

## 스택

바닐라 JS + Canvas 2D. 프레임워크/빌드/백엔드 없음.

## 구조

```
vs-arena/
├── index.html    # 마크업 + UI 골격
├── style.css     # 다크 테마 스타일
├── game.js       # 물리 엔진 + 게임 루프 + UI 바인딩 (전부)
└── BLUEPRINT.md  # 시장분석/BM/보안/로드맵
```

## 보안 노트

- URL 파라미터는 화이트리스트/클램프로 검증 (XSS 차단: DOM 주입은 textContent만 사용)
- 수집하는 데이터 없음 (로그인/개인정보/쿠키 無)
